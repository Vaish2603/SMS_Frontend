

# School Management System - Frontend Client

This repository contains the responsive single-page web client dashboard interface for the School Management System, built using React and styled with modular custom CSS.

## 🚀 Key Modules Included
- **Dashboard Summary Analytics**
- **Student Management (CRUD)**
- **Teacher Management (CRUD)**
- **Daily Attendance Sheet Manifest**
- **Leave Requests Board Workflow**
- **Bulletins Notice Board & Events Tracker**

## 🛠️ Tech Stack
- React.js
- Axios (API Management Client)
- Lucide React (Icon assets pack)
- React Router DOM (Navigation mapping)

## 💻 Local Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/school-system-frontend.git](https://github.com/your-username/school-system-frontend.git)
   cd school-system-frontend

2. Install dependencies:
- >> npm install

3. Configure environment variables:
- >> cp .env.example .env

4. Fire up the development client:
- >> npm run dev
Open your browser and direct it to the output local development server path link.

# 3. Resolving Merge Conflicts

- Open the file in VS Code.
- Locate the conflict markers (<<<<<<< HEAD, =======, >>>>>>>).
- Manually edit or select Accept Current Change, Accept Incoming Change, or Accept Both.
- Save the file and clear the conflict status:
